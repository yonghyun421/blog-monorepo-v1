# My Blog 2026

Next.js 16과 Contentlayer2를 활용하여 구축한 개인 기술 블로그 겸 포트폴리오 사이트입니다.

## 주요 기능

- **MDX 기반 콘텐츠 관리**: Contentlayer2를 통한 타입 안전한 콘텐츠 처리
- **반응형 디자인**: 모바일부터 데스크톱까지 최적화된 레이아웃
- **다크/라이트 모드**: next-themes를 활용한 테마 전환
- **카테고리 & 태그 시스템**: 게시물 분류 및 탐색 기능
- **Bento Grid 레이아웃**: 시각적으로 매력적인 블로그 목록

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: Contentlayer2 + MDX
- **Monorepo**: Turborepo + pnpm
- **Deployment**: Vercel
